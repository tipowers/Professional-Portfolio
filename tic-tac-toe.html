Tic-Tac-Toe Using Python

Try it. You'll have fun!

Click Here To See Github Code

I learned a lot of Python in college and built close to 40 small programs. One of the earlier programs was a game of Rock, Paper, Scissors. We had just learned using nested if-else and if-elif-else statements. Using procedural programming, and a lot of those if statements, I was able to make the simple game of Rock, Paper, Scissors.

The logic of Rock, Paper, Scissors is straightforward and that is how I designed and wrote the code. The player is asked for an input of either 'rock', 'paper', or 'scissors'. If the option is invalid the player is asked to provide input until said input is valid. Once a valid option is input the opponent, in our case 'the computer', generates a random number between 1 and 3.

If the random number is 1 the computer stores the variable as 'rock'. If the random number is 2, 'paper' is the stored variable, or if 3 is randomly generated 'scissors' becomes the computers stored variable. Once the computer pick is randomly generated the user value and computer value are compared to conditional statements that determine the winner based on the result.
