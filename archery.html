For my first attempt at creating a website, that was not 'Hello World', I went with something that I love doing in my free time, Archery. I wanted to create a basic website that introduced users to why I love shooting bows, what kind of bows there are, what components make up the various bows, and videos on how to shoot a bow for those interested.

There was admittedly quite a fair amount of work to be done to the site then, but I was just learning HTML and CSS and how to make them work together. I decided to redo the archery site now that I have more development knowledge and experience. My first redo attempt, also the current one, was to see if I could make better the code already in place, but I have since decided to start the website over from scratch.

Follow me on Github to get informed on site updates.
